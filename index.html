GitHub Publish Setup

A one-time setup. Your token is saved on this device only and never shared.

How to get your token (do this once):
1. Go to github.com and sign in
2. Click your profile picture (top right) → Settings
3. Scroll down and click Developer settings (bottom of left menu)
4. Click Personal access tokens → Tokens (classic)
5. Click Generate new token (classic)
6. Give it a name e.g. HymnDesk Publish
7. Under Select scopes, tick repo (the top checkbox)
8. Scroll down and click Generate token
9. Copy the token shown and paste it below — you will only see it once

Your token is stored only in this browser. If you clear browser data, you will need to paste it again.
